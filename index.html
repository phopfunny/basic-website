ไตรภพ สุขเอี่ยม

นักศึกษาด้านไอที | ผู้พัฒนาแอปพลิเคชัน | นักออกแบบเว็บไซต์

1. ประวัติส่วนตัวและการศึกษา

• ชื่อ: ไตรภพ สุขเอี่ยม

• แผนการเรียน: วิทย์-คณิต

• ผลการเรียน: GPA 3.37

• กิจกรรมที่เกี่ยวข้องกับไอทีในโรงเรียน:

  • เป็นผู้ช่วยดูแลการอบรมด้านตัดต่อและ 3D
  • เข้าอบรมการสร้าง AI

2. แรงบันดาลใจและความสนใจด้านไอที

• เริ่มสนใจไอทีจากการเล่นโปรแกรม Scratch และเขียนโค้ด Python ตอน ม.ต้น

• ชอบการแก้ปัญหา การออกแบบ และการใช้เทคโนโลยีสร้างสิ่งใหม่

• มองว่าเทคโนโลยีสามารถเปลี่ยนแปลงสังคมและคุณภาพชีวิตของผู้คนได้

• ความฝัน: พัฒนาแอปพลิเคชันเพื่อการศึกษาและช่วยเหลือผู้อื่น

• “ผมเชื่อมั่นว่าการศึกษาด้านไอทีจะเปิดโอกาสให้ผมได้สร้างสรรค์สิ่งใหม่ ๆ ที่เป็นประโยชน์ และช่วยพัฒนาคุณภาพชีวิตของผู้คนให้ดีขึ้น”

3. ผลงานด้านไอที

• MyStudyPlanner – เว็บช่วยวางแผนอ่านหนังสือ

• Math Battle – เกมฝึกคณิตศาสตร์ด้วย Scratch

• ระบบลงทะเบียนกิจกรรมโรงเรียน – เชื่อมต่อกับ Google Sheet ด้วย Google Apps Script

• NoteMate (Prototype) – แอปบันทึกและเตือนความจำ

• แนบ GitHub Link / QR Code / ภาพหน้าจอ เพื่อแสดงผลงาน

4. ทักษะและเครื่องมือที่ใช้ได้

• ภาษาโปรแกรม: HTML, CSS, JavaScript, Python (พื้นฐาน), Scratch

• เครื่องมือ: Visual Studio Code, Arduino, Canva, Git (พื้นฐาน), Figma (ออกแบบ UI)

• ทักษะเพิ่มเติม: ออกแบบเว็บไซต์ Responsive, เขียนโค้ดเพื่อแก้ปัญหาในชีวิตจริง

5. ค่าย / กิจกรรม / การอบรมที่เกี่ยวข้อง

• เข้าค่ายเขียนโปรแกรมเบื้องต้นจาก มหาวิทยาลัยเเม่โจ้

• เรียนคอร์สออนไลน์ เช่น “Introduction to Computer Science” (ThaiMOOC), “Build Your First Website” (Coursera)

• เข้าร่วมประกวดโครงงานในงานวิทยาศาสตร์

• ช่วยจัดกิจกรรมแข่งขันเกมและ Coding Game ภายในโรงเรียน
